# GGR472 Final Project
This repository contains codes and data used to create our website, 

# Repository content
- Data from geojson.io/Bicycle Parkng Racks Data - 4326.geojson: Data file containing point location of bicycle parking racks in the city of Toronto
- Data from geojson.io/Cycling-network - 4326-2.geojson: Data file containing line locations of cycle network in the city of Toronto
- Data from geojson.io/Parks and Recreation Facilities - 4326.geojson: Data file containing point locations of parks and recreation facilities in the city of Toronto
- Data from geojson.io/Parks Drinking Fountains - 4326.geojson: Data file containing point locations of public water fountains in the city of Toronto
- Data from geojson.io/Washroom Facilities - 4326-3.geojson: Data file containing point locations of public washrooms in the city of Toronto
- data/bike.gif: GIF image used for the background of index.html title
- data/bikerack.webp: Image used for the background of amenities.html title
- data/peoplecycling.png image used for the information section of amenities.html
- index.html: HTML file for the main page of our website
- script.js: Our java script file that contains the interactivity and turf.js for our web map
- webmap.html: HTML file to render our map
- amenities.html: HTML file for the more information web page
- style.css: CSS file for the positioning of map interface as well as website components
