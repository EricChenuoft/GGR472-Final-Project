How to Navigate the Map

Panning and Zooming

Click and drag the map to move around. Scroll up to zoom in and scroll down to zoom out ( or use the on-screen keys). You can also double-click anywhere on the map to zoom into that spot. Use the fullscreen icon to control map size.

Toggling Map Layers

Use the layer panel to select desired bike route length.

Marker

Use the marker tool to add a marker based on user locaiton to the map
